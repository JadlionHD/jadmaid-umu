# Requirements
1. NodeJS v12 Above
2. npm or yarn
3. `node-gyp`

To install node-gyp you need to is:
```bash
$ npm i node-gyp -g
# or
$ yarn add node-gyp -g
```
After everything has been done, to start that bot you need to do is:
```bash
$ npm install
$ npm start
# or
$ yarn install
$ yarn start
```
